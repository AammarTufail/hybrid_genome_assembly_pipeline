## 1. Overview

This workflow performs an all-vs-all pairwise comparison of plasmid sequences generated by different plasmid prediction approaches. Sequence similarity is calculated using the **MUMmer4** package and its **dnadiff** utility, which combines whole-sequence alignment using `nucmer` with summary statistics describing sequence identity and alignment coverage.


For every plasmid pair originating from different prediction methods, the pipeline computes:

- Average nucleotide identity (ANI)
- Fraction of the reference sequence aligned
- Fraction of the query sequence aligned

The resulting similarity matrix can be used to evaluate agreement between plasmid prediction methods and identify highly similar plasmid reconstructions.

> Link for MUMmer github repository: [https://github.com/mummer4/mummer](https://github.com/mummer4/mummer)

## 3. Pairwise Plasmid Comparison

The workflow compares every plasmid FASTA file against all other plasmid FASTA files.


### Input
For each plasmid you want to compare place the fna file in the same parent directory:

‚îú‚îÄ‚îÄ bakta_sample02.fna
‚îî‚îÄ‚îÄ ...

Each FASTA is expected to be named as:
```text
<approach>_<sample>.fna
```



### Run Analysis

```bash
cd mummer_out || exit 1

# create header for result file
echo "Approach_A,Sample_A,Approach_B,Sample_B,identity,aligned_fraction_ref,aligned_fraction_query" > results.csv

for A in plasmid_dir/*.fna; do
  for B in plasmid_dir/*.fna; do

    # Define approach and sample for both
    base_A=$(basename "$A" .fna)
    base_B=$(basename "$B" .fna)

    approach_A=${base_A%%_*}
    sample_A=${base_A#*_}

    approach_B=${base_B%%_*}
    sample_B=${base_B#*_}

    prefix="${base_A}__${base_B}"

    # skip comparisons from the same prediction approach
    if [ "$approach_A" = "$approach_B" ]; then
      continue
    fi

    # run dnadiff for pairwise comprison
    dnadiff -p "$prefix" "$A" "$B" > /dev/null 2>&1

    # include results into summary csv file
    report="${prefix}.report"

    identity=$(awk '/AvgIdentity/ {for(i=1;i<=NF;i++) if($i ~ /^[0-9.]+$/) v=$i} END{print v}' "$report")
    ref_aligned=$(awk '/AlignedBases/ {print $2}' "$report")
    qry_aligned=$(awk '/AlignedBases/ {print $3}' "$report")
    ref_total=$(awk '/TotalBases/ {print $2}' "$report")
    qry_total=$(awk '/TotalBases/ {print $3}' "$report")
    frac_ref=$(awk -v a="$ref_aligned" -v t="$ref_total" 'BEGIN{print (t>0 ? a/t : 0)}')
    frac_qry=$(awk -v a="$qry_aligned" -v t="$qry_total" 'BEGIN{print (t>0 ? a/t : 0)}')

    printf "%s,%s,%s,%s,%s,%s,%s\n" \
      "$approach_A" "$sample_A" \
      "$approach_B" "$sample_B" \
      "$identity" "$frac_ref" "$frac_qry" >> results.csv

  done
done

cp results.csv ../mummer_results.csv 
```

### Output

A combined CSV file containing pairwise similarity metrics:

```text
results.csv
```

Each row represents a single pairwise plasmid comparison.

## 4. Similarity Metrics

The following metrics are extracted from each `dnadiff` report:

| Metric | Description |
|----------|-------------|
| `identity` | Average nucleotide identity (%) |
| `aligned_fraction_ref` | Fraction of the reference plasmid covered by the alignment |
| `aligned_fraction_query` | Fraction of the query plasmid covered by the alignment |
